# Hero Search - React

Aplicação desenvolvida em React que consome a SuperHero API para buscar personagens pelo nome.

## Tecnologias
- React
- JavaScript
- Fetch API
- Vite

## Funcionalidades
- Busca de heróis por nome
- Tratamento de erro
- Estado de loading
- Separação de responsabilidades (service layer)

## Como rodar o projeto

npm install
npm run dev
